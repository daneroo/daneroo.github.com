  • Linux: download iperf-1.7.0-linux-2.3.tar.gz ; glibc-2.3 executable, untar, and run
  • Windows: download iperf-1.7.0-win32.exe ; windows self-extracting zip, unzip and run
Simple:
Server side: ./iperf -s
Client side: ./iperf -c

Explicit:
./iperf -s -P 0 -i 1 -p 5001 -l 8K -f m -t 10
./iperf -c
